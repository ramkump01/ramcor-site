# RAMCOR Website

A polished single-page website for RAMCOR, featuring IT services, financial advisory, AI and software development services, and a contact form that opens the user’s email client with a prefilled message for ramkump01@googlemail.com.

## Deploy to Vercel

1. Push this folder to GitHub.
2. Sign in to Vercel and import the repository.
3. Vercel will deploy the site automatically as a static site.

## Local preview

Open index.html in your browser, or serve the folder with a simple local server.
